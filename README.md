# Nexus Media Controller

## System Requirements

- **Operating System**: Windows
- **Python Version**: Python 3.7 or higher

## Installation

1. Navigate to the project folder (Download the project files or clone using git):
   ```bash
   git clone https://github.com/Ic3Xer/Computer-Music-Player.git
   ```

2. Install the required external libraries:
   ```bash
   pip install -r requirements.txt
   ```

## How to Start

1. Run the main server file:
   ```bash
   python ComMSP.py
   ```

2. Observe the Python console output to find the IP address for connection.

3. Open your web browser and navigate to the IP address provided.
