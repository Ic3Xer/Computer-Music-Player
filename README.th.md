# COM Media Controller

## ความต้องการของระบบ

- **ระบบปฏิบัติการ**: Windows
- **เวอร์ชัน Python**: Python 3.7 ขึ้นไป

## วิธีการติดตั้ง

1. เข้าไปยังโฟลเดอร์ของโปรเจกต์:
   ```โหลด ไฟล์ Project นี้หรือ
   git clone https://github.com/Ic3Xer/Computer-Music-Player.git
   ```

2. ติดตั้งไลบรารีภายนอกที่จำเป็น:
   ```bash
   pip install -r requirements.txt
   ```

## วิธีเริ่มใช้งาน

1. รันไฟล์เซิร์ฟเวอร์หลัก:
   ```bash
   python ComMSP.py
   ```

2. สังเกตที่หน้าตอนรัน Python เพื่อดูไอพีสำหรับเชื่อมต่อ

3. เปิดเบราว์เซอร์ผ่าน IP ที่ได้ไป
